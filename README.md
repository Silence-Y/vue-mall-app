# vue-mall-app  
## 前言  
基于Vue + Vant开发的每日优鲜App项目  
## 技术栈
- Vue  
- Vant
- 本地化存储
## 功能实现  
- 商品浏览  
- 商品搜索
- 查看历史搜素记录，本地化存储
- 加入购物车
- 购物车结算

## 运行  
1.安装依赖
```
npm install
```

2.本地运行
```
npm run serve
```

3.编译打包
```
npm run build
```
